# Ntsiniz Growth + ASO Playbook (International)

## Positioning

Ntsiniz is **training**, not karaoke:

**“Pitch Lock training: listen → match → record → feedback → save best take → next drill.”**

### Primary keywords

* learn to sing
* vocal coach
* pitch trainer
* pitch accuracy
* ear training

## Store listing (screenshot storyboard)

1) **Listen & Match** (mandatory reference + “Ready…”)
2) **Live pitch guidance** (no jank, clear target)
3) **Playback + waveform seek**
4) **Save best take** (progress proof)
5) **Today’s plan (3 drills)**
6) **Weekly report (proof of improvement)**
7) **7‑Day Pitch Lock challenge** (habit)
8) **Mic test + Studio guidance** (trust)

## Reviews engine

Ask for review only after:
* First “win” (high score + session completed)
* Day‑3 or Day‑7 streak milestone

Never ask on first open.

## Viral loop (no social network required)

Shareable artifacts:
* Weekly progress card
* Challenge badge (X/7 days)
* Best take card (audio + trace in future)
