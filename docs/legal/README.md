## Publishing notes

- These docs are written to be launch-safe without placeholder markers.
- Before going live, ensure your store listing includes an accurate contact method (Support link/email) and that it matches the in-app Support path.
- Confirm third-party services enabled (analytics, crash reporting, storage providers).
- Confirm whether minors are allowed and add parental consent rules if needed.
- Review copyright policy for covers/backing tracks.
- Have a qualified lawyer review.
