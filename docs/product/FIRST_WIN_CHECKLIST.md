# First win checklist (launch critical)

The first 2 minutes decide retention.

## The first drill must:
- be easy enough that most users succeed
- show ONE clear win moment (e.g., “You held the note stable for 1.2s ✅”)
- show “What next?” instantly

## Avoid:
- multiple stats before a win
- long explanations
- early paywall interruptions

## UX rules
- Every screen answers:
  1) What is this?
  2) What’s next?
  3) What happens if I do it?

## QA pass
- Cold start → first drill started in < 15 seconds
- First drill completion shows:
  - success moment
  - next action button
