# Ntsiniz — Store Listing Draft (v1)

Use this as a **starting point** for App Store Connect + Google Play Console.

## App name
Ntsiniz

## Tagline / short description
Sing with a Ghost Guide. Train pitch, timing, and confidence.

## What it is (1–2 lines)
Ntsiniz is a vocal training app that turns practice into a fun, game-like experience with an **Aurora Ghost Guide** — hit the lane, hold the note, build streaks.

## Key features (bullets)
- **Ghost Guide (Aurora)**: a Guitar‑Hero‑style pitch highway that shows where to aim.
- **Instant coaching**: “Why + Fix” moments after each take + a 30‑second micro‑drill.
- **Phrase grading**: Perfect / Clean / Almost with clear, friendly feedback.
- **Duets**: share a duet pack, record your reply, and post.
- **Competitions**: seasons, rounds, submissions, leaderboards.
- **Coach marketplace**: programs + feedback inbox, including offline coach workflow.
- **Safety tools**: reporting + moderation audit log.

## Description (store long description)
Train your voice with clarity, not anxiety.

Ntsiniz helps you sing in tune and on time using a beautiful **Aurora Ghost Guide** that makes practice feel like a game.

### How it works
1) Pick a drill or program day.
2) Sing along with the Ghost Guide lane.
3) Get a simple phrase grade (Perfect / Clean / Almost) and a clear “What now?” fix.

### Built for real progress
- Targeted drills for pitch stability, intervals, resonance, vibrato, agility
- Explainable grading and next‑step coaching
- A calm, modern UI that keeps you moving forward

### Optional Pro
Unlock advanced Ghost Guide insights and premium training content. Cancel anytime.
