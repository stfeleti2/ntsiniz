# Go-live checklist (Items 1–7)

This folder contains the “last mile” items required before App Store / Play Store submission.

## 1) Terms + Privacy pages
- In-app **Privacy** and **Terms** screens exist (Settings → Privacy/Terms).
- For store submission, you must also host public URLs and set `EXPO_PUBLIC_PRIVACY_URL` and `EXPO_PUBLIC_TERMS_URL`.
